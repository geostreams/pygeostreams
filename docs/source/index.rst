Pygeotemporal
=============

Pygeotemporal is a Python library to simplify interacting with `Clowder <https://clowder.ncsa.illinois.edu/>`_, an open source
data management system for research data and `Geo Temporal API V2 <https://opensource.ncsa.illinois.edu/bitbucket/projects/GEOD/repos/geo-temporal-api-v2>`
